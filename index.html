Sicherer Code Generator

Erzeuge Codes mit 12 bis 16 Zeichen und sieh Schritt fuer Schritt, wie aus Zeichensaetzen ein zufaelliger Code entsteht.

Einstellungen

Ergebnis

Klicke auf Generieren

So funktioniert die Generierung

  1. 1

    Zeichensaetze werden nach deinen Haken ausgewaehlt.

  2. 2

    Die Laenge wird zwischen 12 und 16 Zeichen festgelegt.

  3. 3

    Aus jedem aktivierten Zeichensatz wird mindestens ein Zeichen genommen.

  4. 4

    Die restlichen Zeichen werden zufaellig ergaenzt.

  5. 5

    Alles wird gemischt, damit der Code nicht vorhersehbar ist.

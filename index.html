Sicherer Code Generator

Erzeuge Codes mit 12 bis 16 Zeichen und sieh Schritt für Schritt, wie aus Zeichensätzen ein zufälliger Code entsteht.

So funktioniert die Generierung

  1. 1

    Zeichensätze werden nach deinen Haken ausgewählt.

  2. 2

    Die Länge wird zwischen 12 und 16 Zeichen festgelegt.

  3. 3

    Aus jedem aktivierten Zeichensatz wird mindestens ein Zeichen genommen.

  4. 4

    Die restlichen Zeichen werden zufällig ergänzt.

  5. 5

    Alles wird gemischt, damit der Code nicht vorhersehbar ist.
